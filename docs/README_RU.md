# 🛡️ Система контроля безопасности — Детекция опасных объектов и подсчёт людей

> **Тема проекта:** Создание программы, которая автоматически находит и считает объекты на изображениях для использования в контроле качества (уведомления об опасности).

---

## 📋 Описание

Данная программа использует **компьютерное зрение (OpenCV)** и модели **Ultralytics YOLOv8** для автоматического обнаружения:

- 👤 **Людей** — подсчёт количества людей на изображении
- 🦴 **Скелета (pose)** — ключевые точки человека (YOLOv8-pose)
- 📦 **Объектов из COCO (80 классов)** — по списку `models/coco.names`
- 🔪 **Опасных предметов** (MVP) — нож, ножницы, бита
- ⚠️ **Опасных действий** (MVP) — руки вверх (HANDS UP)

При обнаружении опасных объектов система автоматически генерирует **визуальное уведомление об угрозе**.


```bash

```

## 📖 Как работает программа

### Алгоритм работы

1. **Загрузка изображения** → в формате BGR через OpenCV
2. **Предобработка** → нормализация (0–1), преобразование в blob (416×416)
3. **Прямой проход нейронной сети** → YOLOv4-tiny обрабатывает blob
4. **Фильтрация** → отбор детекций по порогу уверенности
5. **NMS (Non-Maximum Suppression)** → устранение дублирующих рамок
6. **Классификация угроз** → проверка классов (knife, scissors, bat)
7. **Визуализация** → отрисовка рамок, подписей, панели предупреждений
8. **Warm-up** → предварительный прогон пустого изображения для ускорения

### Параметры детекции

| Параметр | Значение | Описание |
|---|---|---|
| Порог уверенности | 35% (настраивается) | Минимальная уверенность детекции |
| NMS‑порог | 40% | Подавление перекрывающихся рамок |
| Размер входа | 416×416 | Размер blob для нейронной сети |

## 🧠 О моделях YOLOv8 (Ultralytics)

**YOLO (You Only Look Once)** — архитектура нейронной сети, способная обнаруживать объекты за один проход, что обеспечивает высокую скорость работы.

Используются лёгкие модели для MVP (скорость на CPU выше, качество чуть ниже):

- `yolov8n.pt` — детекция объектов (COCO 80 классов)
- `yolov8n-pose.pt` — поза/скелет человека + ключевые точки

Модели запускаются через библиотеку **ultralytics**.
